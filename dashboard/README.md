# NCVS Dashboard

This is the code I used to prep the NCVS dashboard. I downloaded the [data from ICPSR](https://www.icpsr.umich.edu/web/NACJD/studies/39273), and you can see in `prep_parquet_data.R` how I created the dataset.

`AntigravityPrompt.txt` is the original prompt I used for the Antigravity coding tool to create the R code (and initial dashboard).

The final webapp is available at <https://crimede-coder.com/graphs/ncvs.php>
